Dokumentasi API


Blog Digital menyediakan RESTful API (Application Programming Interface) yang dapat Anda pakai untuk membuat berbagai macam aplikasi yang sedang Anda bangun. RESTful API yang kami buat mudah untuk diintegrasikan karena menggunakan arsitektur REST dengan format balasan berupa JSON yang didukung oleh hampir semua bahasa pemrograman.
Sudah dapat ide untuk membuat aplikasi dengan RESTful API ini? Silakan gunakan dengan baik, jika menemukan salah satu API error, segera hubungi kami agar segera kami fix errornya.

API url

Ini adalah Base Url API kami, silakan simpan di notepad terlebih dahulu dengan mengklik icon copy untuk menyalinnya.

Parameter

Parameter Keterangan
key key dapat Anda lihat di Daftar API, tiap API memiliki key berbeda tergantung fungsi dari tiap API
action action dapat juga Anda lihat di Daftar API, action berfungsi untuk memberikan Anda kunci untuk menggunakan API sesuai tipenya.
url, text, query, search, username, id Paramater ini bersifat formal, tiap API memiliki paramater yang berbeda beda, bisa url, text, username dan lain lain tergantung kegunaannya API tersebut. Anda dapat melihatnya di Daftar API.
